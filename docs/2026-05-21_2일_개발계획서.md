```
필수 적용:
- 시스템 프롬프트에 grounded_only 지침 포함
- 문서에 없는 내용은 "제공된 문서에서 해당 정보를 찾을 수 없습니다" 응답
```

### 7.2 기존 API 모드 활용

```python
# /api/rag/query의 mode 파라미터 활용
mode 값:
```
