# Phase 8. admin.html UI 설계서

## 1. 문서 개요

이 문서는 admin.html 관리자 화면의 구조, 탭 구성, 설정 UI를 설계한다.
Phase 6에서 정의한 파이프라인을 실행하고 모니터링하는 UI를 정의한다.

## 2. 탭 구조

### 2.1 전체 탭 목록

```
┌─────────┬─────────┬──────────┬─────────────────┬───────────┐
│Dashboard│Documents│ Metadata │ RAG Build Wizard│ Graph RAG │
├─────────┴─────────┴──────────┴─────────────────┴───────────┤
│         (기존)         │           (신규)                   │
├─────────┬─────────┬──────────┬─────────────────┬───────────┤
│ LLM Wiki│  Jobs   │  Logs    │    Settings     │  Search   │
│         │         │          │                 │  Quality  │
└─────────┴─────────┴──────────┴─────────────────┴───────────┘
```

| 탭 | 상태 | 설명 |
| --- | --- | --- |
| Dashboard | 기존 | 시스템 현황 대시보드 |
| Documents | 기존 | 문서 목록 및 관리 |
| Metadata | 기존 | 메타데이터 확인 및 확정 |
| RAG Build Wizard | **신규** | RAG 파이프라인 실행 |
| Graph RAG | **신규** | Graph RAG 빌드 및 시각화 |
| LLM Wiki | **신규** | Wiki 생성 및 관리 |
| Jobs | **신규** | Job 목록 및 모니터링 |
| Logs | **신규** | 처리 로그 및 에러 |
| Settings | **신규** | 시스템 설정 |
| Search Quality | **신규** | 검색 품질 테스트 |
