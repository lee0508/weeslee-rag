# 1. 현재 화면 분석

## 1.1 현재 페이지의 장점

현재 페이지에는 문서 기반 AI Agent로 발전할 수 있는 핵심 기능이 이미 포함되어 있다.

현재 확인된 주요 기능은 다음과 같다.

1. RAG Assistant 화면 존재
2. Knowledge Search 화면 존재
3. Answer Review 화면 존재
4. 제안서 초안 생성 화면 존재
5. General RAG / Bid Project Search / RFP Analysis / Graph RAG 검색 모드 존재
6. Top K, Top Docs, Max Chunks / Doc 설정 존재
7. Category 필터 존재
8. Organization Filter, Year Filter 존재
9. Ollama 기반 답변 생성 모드 존재
10. Collection 선택 기능 존재
11. Vector / Graph / Wiki 검색 비율 설정 존재
12. RFP 파일 첨부 기능 존재
13. 그래프 결과 패널 존재
14. 세션 기록 기능 존재
15. 제안서 생성 섹션 선택 기능 존재

즉, 기능 요소는 상당히 많이 들어가 있다.

문제는 기능이 없는 것이 아니라,
기능이 사용자의 업무 흐름에 맞게 정리되어 있지 않다는 점이다.
