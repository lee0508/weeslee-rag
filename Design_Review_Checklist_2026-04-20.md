## 1. 상단 네비게이션 (Top Navigation)

### 1.1 메뉴 탭 버튼

| 현재 코드 | 확인 필요 사항 |
|----------|---------------|
| `🏠 홈` | ① 클릭 시 이동할 화면 디자인 필요 (대시보드?) |
| `📚 프롬프트 라이브러리` | ① 별도 페이지 디자인 필요 ② 명칭 변경 여부: "작업 템플릿"? |
| `▶ 새 작업 실행` | ① 현재 메인 화면과 동일한지, 별도 진입점인지 ② 명칭 변경 여부: "새 문서 작성"? |
| `📋 실행 이력` | ① **별도 페이지 디자인 필요** (목록형 UI) ② 상세 화면 디자인 필요 |
| `⚙ 관리자` | ① 현재는 모달로 열림 → 별도 페이지로 변경할지? |

### 1.2 사용자 영역

| 요소 | 확인 필요 사항 |
|------|---------------|
| `Beta v1.0` 배지 | 정식 출시 시 표시 방식? 버전 표기 규칙? |
| 아바타 `김` | ① 프로필 이미지 지원 여부 ② 클릭 시 드롭다운 메뉴 필요? (설정, 로그아웃 등) |
| `김위즐 | 컨설팅팀` | ① 부서명 외 역할(권한) 표시 필요? ② 클릭 시 프로필 페이지 이동? |

## 2. 좌측 사이드바 (Left Sidebar)

### 2.1 검색 영역

| 요소 | 확인 필요 사항 |
|------|---------------|
| 검색 입력창 | ① placeholder "프롬프트 검색..." → "템플릿 검색..."으로 변경? ② 검색 범위: 이름만? 설명 포함? 태그 포함? |
| 🔍 아이콘 | 클릭 시 동작? (현재는 장식용) |

### 2.2 필터 칩

| 현재 칩 | 확인 필요 사항 |
|--------|---------------|
| `전체` `보고서` `질의응답` `요약` `분석` | ① 카테고리 목록 최종 확정 필요 ② 다중 선택 가능 여부? ③ 칩 추가/편집 기능 필요? |

### 2.3 템플릿 그룹

| 그룹 | 확인 필요 사항 |
|------|---------------|
| `⭐ 즐겨찾기` | ① 즐겨찾기 추가/제거 방법 (드래그? 버튼?) ② 빈 상태 디자인 |
| `🏛 조직 표준 템플릿` | ① 접기/펼치기 기능 필요? ② 항목 수가 많을 때 스크롤 vs 더보기 |
| `👤 개인 저장` | ① 개인 템플릿 최대 개수 제한? ② 삭제 기능 위치 |
| `🕐 최근 사용` | ① 표시 개수 (현재 2개 고정?) ② 기간 기준 (7일? 30일?) |

### 2.4 템플릿 카드

| 요소 | 확인 필요 사항 |
|------|---------------|
| 카드 전체 | ① 호버 시 추가 정보 표시? (사용 횟수, 평점 등) ② 우클릭 컨텍스트 메뉴 필요? |
| 상태 태그 `표준승인` `검토중` `개인안` | ① 색상 규칙 확정 (승인=초록, 검토=노랑, 개인=회색?) ② 추가 상태 필요? (폐기됨, 임시저장 등) |

### 2.5 버튼

| 버튼 | 확인 필요 사항 |
|------|---------------|
| `＋ 새 템플릿 만들기` | 권한 없는 사용자에게 표시 여부? (비활성화 vs 숨김) |
| `나의 실행 이력` | 뱃지 숫자 `12` 기준? (오늘? 전체 미확인?) |
| `결과 보관함` | ① 보관함 용량 제한 표시 필요? ② 뱃지 숫자 기준? |

## 3. 메인 영역 - Step Wizard

### 3.1 단계 구성

| 현재 단계 | 확인 필요 사항 |
|----------|---------------|
| 1. 업무선택 | ① "업무선택" → "문서유형 선택"으로 변경? ② 별도 화면 필요? (현재는 사이드바에서 선택) |
| 2. 프롬프트선택 | ① "프롬프트선택" → "템플릿 선택"으로 변경? ② Step 1과 통합 가능? |
| 3. 입력값 설정 | 현재 메인 화면 |
| 4. 지식원 선택 | ① 별도 화면? 현재는 Right Panel에 있음 ② Step 3과 동시 표시 vs 순차 표시? |
| 5. 결과 검토 | 생성 완료 후 자동 전환? |

### 3.2 Wizard 동작

| 동작 | 확인 필요 사항 |
|------|---------------|
| 단계 클릭 | ① 완료된 단계만 클릭 가능? ② 미완료 단계 클릭 시 경고? |
| 단계 건너뛰기 | 필수 단계 건너뛰기 방지 로직? |
| 진행 표시 | 현재 단계 외에 진행률(%) 표시 필요? |

## 4. 메인 영역 - 템플릿 헤더 카드

### 4.1 메타 정보

| 요소 | 확인 필요 사항 |
|------|---------------|
| 카테고리 태그 `📑 보고서 작성` | 아이콘+텍스트 조합 유지? 카테고리별 색상 규칙? |
| 버전 태그 `v2.3` | 클릭 시 버전 이력 모달? 현재 동작 없음 |
| 상태 태그 `✓ 표준승인` | 상태별 색상 최종 확정 (현재: 초록) |

### 4.2 정보 행

| 항목 | 확인 필요 사항 |
|------|---------------|
| 작성자 | 클릭 시 작성자 프로필/다른 템플릿 보기? |
| 최종 수정 | 시간까지 표시 필요? (현재: 날짜만) |
| 사용 횟수 | 기간 기준? (전체 누적 vs 최근 30일) |
| 평균 만족도 | 평가 개수 함께 표시? (예: ⭐ 4.3 (48명)) |

### 4.3 버튼

| 버튼 | 확인 필요 사항 |
|------|---------------|
| `📋 템플릿 전체보기` | 모달 vs 별도 페이지? |
| `🕒 버전이력` | 버전 비교(diff) 기능 필요? |
| `📊 활용통계` | ① 어떤 통계 항목 표시? ② 기간 필터 필요? |

## 5. 메인 영역 - 입력 파라미터

### 5.1 필드 구성

| 필드 | 확인 필요 사항 |
|------|---------------|
| 기관명 * | ① 자동완성(기존 입력값) 필요? ② 기관 DB 연동? |
| 사업명 * | 자동완성 필요? |
| 작성 목적 * (select) | ① 옵션 목록 최종 확정 ② "기타" 직접 입력 옵션? |
| 독자 수준 (select) | 옵션 목록 최종 확정 |
| 출력 형식 (select) | 옵션 목록 최종 확정 |
| 분량 설정 (select) | ① 옵션 값 (3p, 5p, 10p, auto) 적절? ② 직접 입력 옵션? |
| 핵심 키워드 | ① 태그 형식 입력? (쉼표 구분 vs 개별 태그) ② 추천 키워드 기능? |
| 제외 조건 | 위와 동일 |
| 대상 시스템 / 정책명 | 위와 동일 |

### 5.2 도움말 버튼 `?`

| 필드 | 확인 필요 사항 |
|------|---------------|
| 각 필드별 `?` 버튼 | ① 모달 vs 툴팁? ② 모든 필드에 필요? 필수 필드만? |

### 5.3 헤더 버튼

| 버튼 | 확인 필요 사항 |
|------|---------------|
| `👁 프롬프트 미리보기` | 실시간 업데이트 vs 버튼 클릭 시 갱신? |
| `🗑 초기화` | 확인 다이얼로그 필요? |

### 5.4 프롬프트 미리보기 영역

| 요소 | 확인 필요 사항 |
|------|---------------|
| 미리보기 박스 | ① 높이 고정 vs 내용에 따라 확장? ② 스크롤 표시 방식? |
| 변수 하이라이트 | 미입력 변수 색상 (현재: amber) 확정? |
| `전체보기 →` 버튼 | 모달로 열기 확정? |

## 7. 메인 영역 - AI 생성 결과

### 7.1 헤더 영역

| 요소 | 확인 필요 사항 |
|------|---------------|
| 모델 뱃지 `claude-sonnet` | 사용자가 변경 가능? 클릭 시 모델 설정? |
| `⭐` 버튼 | 평가 완료 후 상태 변화? (채워진 별?) |
| `📋` 복사 버튼 | 복사 성공 피드백 (툴팁? 토스트?) |
| `⬇` 다운로드 버튼 | 바로 다운로드 vs 포맷 선택 모달? |

### 7.2 진행 상태

| 요소 | 확인 필요 사항 |
|------|---------------|
| 프로그레스 바 | ① 색상 (현재: navy→amber 그라데이션) 확정 ② 취소 버튼 필요? |
| 단계 텍스트 | 표시할 단계 목록 확정 (지식원 검색 → 청크 추출 → 프롬프트 구성 → LLM 응답) |

### 7.3 결과 영역

| 상태 | 확인 필요 사항 |
|------|---------------|
| 빈 상태 | 현재 디자인 확정? (✦ 아이콘 + 안내 문구) |
| 생성 중 | 스켈레톤 UI? 로딩 스피너? (현재: 스피너) |
| 결과 표시 | ① 최대 높이 제한? 스크롤? ② 출처 표시 스타일 (현재: amber 배경 인라인) |
| 에러 상태 | **디자인 없음** - 필요 |

### 7.4 결과 액션 버튼

| 버튼 | 확인 필요 사항 |
|------|---------------|
| `🔄 다시 생성` | ① 동일 입력으로 재생성? ② 확인 다이얼로그? |
| `✏️ 부분 수정 요청` | ① 텍스트 선택 후 수정? ② 모달에서 수정 내용 입력? **상세 UX 필요** |
| `📊 표로 변환` | 전체 변환? 선택 영역만? |
| `📤 내보내기` | 모달 열기 확정 |
| `💾 결과 저장` | 모달 열기 확정 |
| `⬆ 템플릿 개선 요청` | ① 관리자에게 전달? ② 요청 내용 입력 폼 필요? **모달 디자인 없음** |

### 7.5 평가 영역

| 요소 | 확인 필요 사항 |
|------|---------------|
| 별점 (5점) | ① 0.5점 단위? 1점 단위? ② 평가 후 수정 가능? |
| `📝 상세 피드백` 버튼 | 평가 완료 후에만 활성화? |
