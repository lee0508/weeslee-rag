**목적**: organization / project_name 추출 신뢰도 — 나중에 graph/wiki 오류 방지

**현재 구현**: `enrich_project_metadata()` → project_name (규칙 기반, 날짜 prefix 제거)

**추가할 로직** (`extract_manifest_batch.py`):

```python
def enrich_project_metadata(folder_name: str) -> dict:
    project_name = _DATE_PREFIX.sub("", folder_name).strip()
    year_match = re.match(r"^(\d{4})", folder_name)
    folder_year = year_match.group(1) if year_match else ""

    # 신뢰도: 날짜 prefix가 있으면 높음 (규칙 기반 추출)
    project_confidence = 0.90 if year_match else 0.50

    return {
        "project_name": project_name,
        "project_confidence": project_confidence,
        "folder_year": folder_year,
        "folder_name": folder_name,
    }
```
