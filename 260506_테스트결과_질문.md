현재 메시지 의미:

RFP 100% pass
Proposal 100% pass
final_report 0 docs
presentation 0 docs

즉 rfp, proposal 인덱스는 정상 검색되지만, final_report, presentation 전용 인덱스를 API/검색 스크립트가 못 찾고 있습니다.

가장 가능성 높은 원인입니다.

1. 실제 생성 파일명과 코드에서 찾는 파일명이 다름
2. category 이름 불일치
   - final_report vs final-report vs finalreport
   - presentation vs ppt/pptx
3. .env의 FAISS_INDEX_PREFIX가 이전 combined 인덱스를 가리킴
4. category index manifest에는 있는데 metadata jsonl 경로가 다름
5. build_category_indexes.py가 final_report/presentation 파일을 생성하지 못함

바로 서버에서 확인할 명령어입니다.

cd /data/weeslee/weeslee-rag

echo "=== category index files ==="
find data/indexes/faiss -maxdepth 1 -type f | sort | grep -E "category|rfp|proposal|final|presentation|kickoff"

echo "=== env ==="
grep -E "FAISS|INDEX|CATEGORY" .env backend/.env 2>/dev/null

echo "=== metadata category counts ==="
python3 - <<'PY'
import json, collections, glob

files = glob.glob("data/indexes/faiss/*combined*metadata*.jsonl")
print(files)

for fp in files:
    c = collections.Counter()
    with open(fp, encoding="utf-8") as f:
        for line in f:
            if not line.strip():
                continue
            row = json.loads(line)
            c[row.get("category") or row.get("document_type") or "NONE"] += 1
    print(fp)
    print(c)
PY

그리고 category index 파일 자체 확인:

python3 - <<'PY'
import faiss, glob, os

for fp in sorted(glob.glob("data/indexes/faiss/*.index")):
    if any(x in fp for x in ["rfp", "proposal", "final", "presentation", "kickoff"]):
        idx = faiss.read_index(fp)
        print(os.path.basename(fp), "vectors=", idx.ntotal, "dim=", idx.d)
PY

정상이라면 이런 식으로 나와야 합니다.

..._category_rfp.index vectors > 0
..._category_proposal.index vectors > 0
..._category_final_report.index vectors > 0
..._category_presentation.index vectors > 0

바로 수정할 부분은 보통 검색 코드의 category → 파일명 매핑입니다.

# category_alias.py 또는 rag_service.py 내부에 추가 권장

CATEGORY_ALIASES = {
    "rfp": "rfp",
    "proposal": "proposal",
    "kickoff": "kickoff",
    "final_report": "final_report",
    "final-report": "final_report",
    "finalreport": "final_report",
    "report": "final_report",
    "presentation": "presentation",
    "ppt": "presentation",
    "pptx": "presentation",
}

def normalize_category(category: str | None) -> str | None:
    if not category:
        return None

    key = category.strip().lower().replace(" ", "_")
    return CATEGORY_ALIASES.get(key, key)
