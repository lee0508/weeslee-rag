# 260507 운영공지 웹서비스 주소 정리

## 목적

회사 서버 `192.168.0.207`의 주요 웹서비스 주소 체계를 정리한 운영 공지이다.

- 기준일: 2026-05-07
- 대상 서버: `192.168.0.207`
- 외부 대표 도메인: `https://server.weeslee.co.kr`

## 기본 원칙

- 외부 서비스는 기본적으로 `https://server.weeslee.co.kr/{service}` 형식으로 사용한다.
- 내부 서비스는 기본적으로 `http://192.168.0.207/{service}` 형식으로 사용한다.
- 주소 입력 시 영문 대소문자는 구분하지 않는다.
- 예외 서비스는 실제 운영 주소를 우선 사용한다.
